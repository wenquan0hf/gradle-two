# 介绍

Gradle 为Java（JVM）世界提供快速构建的工具。提供如下功能：

- 一个非常灵活的通用构建工具,如 Ant
- 方便从 Maven 中切换过来。但我们从不强制
- 对多项目构建非常支持
- 很强的依赖性管理（基于 Apache Ivy）
- 对你现有的 Maven 或者 Ivy 库全力支持
- 支持传递依赖管理，不需要远程仓库或者 pom.xml 和 ivy.xml 文件
- Ant 的任务和构建是一等公民
- Groovy 为构建使用脚本
- 一个丰富的域模型描述你的构建
